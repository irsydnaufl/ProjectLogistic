# Inventory and Delivery Tracking System

Project ini merupakan **sistem monitoring logistik berbasis Excel** yang dirancang untuk **mengelola proses stok barang, pemantauan distribusi pengiriman, serta analisis performa logistik** menggunakan rumus Excel, Pivot Table, dan dashboard interaktif.

- **Sheet Kode Data**<br>
Sheet Kode Data berfungsi sebagai referensi utama (master reference) yang menyimpan daftar kode barang dan kategori barang yang digunakan dalam sistem, serta mengelompokkan barang berdasarkan kategori sehingga dapat menjadi referensi untuk data inventory.<br>
![Uploading Sheet Kode Data.png…](https://github.com/irsydnaufl/ProjectLogistic/blob/main/Sheet/Sheet%20Kode%20Data.png)<br>

- **Sheet Master Data**<br>
Sheet ini merupakan database utama inventory yang mencatat seluruh aktivitas barang masuk dan barang keluar di gudang. Sheet ini digunakan untuk monitoring persediaan barang di gudang, mengetahui jumlah stok yang tersedia, mengontrol barang masuk dan keluar dan menjadi sumber data untuk analisis stok pada Pivot Table.<br>
![Uploading Sheet Master Data.png…](https://github.com/irsydnaufl/ProjectLogistic/blob/main/Sheet/Sheet%20Master%20Data.png)<br>

- **Sheet Tracking Distribusi Barang**<br>
Sheet ini merupakan bagian utama dalam monitoring distribusi logistik karena mencatat seluruh aktivitas pengiriman barang ke customer. Sheet ini membantu staff logistik untuk memantau status pengiriman, mengukur kinerja distribusi, mengidentifikasi keterlambatan pengiriman dan mengevaluasi ekspedisi atau armada pengiriman.<br>
![Uploading Sheet Tracking Distribusi Barang.png…](https://github.com/irsydnaufl/ProjectLogistic/blob/main/Sheet/Sheet%20Tracking%20Distribusi%20Barang.png)<br>

- **Sheet Pivot Tabel**<br>
Sheet ini digunakan untuk melakukan analisis data secara otomatis dari database inventory dan distribusi. Antara lain menampilkan jumlah stok akhir setiap barang di gudang, digunakan untuk mengetahui supplier yang paling banyak memasok bahan, menunjukkan wilayah dengan distribusi barang tertinggi, dll.
![Uploading Sheet Pivot Table.png…](https://github.com/irsydnaufl/ProjectLogistic/blob/main/Sheet/Sheet%20Pivot%20Table.png)<br>

- **Sheet Dashboard**<br>
Sheet Dashboard merupakan tampilan visual utama dari sistem monitoring logistik. Dashboard ini dirancang untuk menampilkan Key Performance Indicators (KPI) yang penting bagi operasional logistik.<br>
![Uploading Dashbord.png…](https://github.com/irsydnaufl/ProjectLogistic/blob/main/Sheet/Dashbord.png)<br>
